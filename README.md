**Социальная сеть для купли-продажи автомобилей с прозрачной историей владельцев.**

---

## Проблема

При покупке автомобиля с рук сложно узнать реальную историю машины. Продавцы скручивают пробег, скрывают количество владельцев, не говорят о ДТП. Покупатель полагается на слова и "опытного дядю". Доверия нет.

## Решение

AutoTrust — платформа, где каждый автомобиль получает цифровой след: кто владел, когда продал, как долго пользовался. История привязана к конкретной машине и не может быть удалена. Покупатель видит полную хронологию владения, продавец получает репутацию.
